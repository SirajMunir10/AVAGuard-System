About AVAGuard Policies

Policies define the compliance rules enforced by AVAGuard agents. Use this configuration to:

  • Define Compliance Rules: Select which CIS, HIPAA, or Custom checks are active during desktop scans.
  • Enforce Standards: Set severity levels (Critical, High, Medium) and pass/fail thresholds.
  • Manage Exceptions: Temporarily disable specific checks or create organization-specific rules.

Note: Policy changes are automatically synchronized to desktop agents during their next scan cycle.
