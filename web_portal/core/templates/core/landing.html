Immutable Hash-Chained Audit Log

Automated CIS Benchmark Assessment

Run 10 automated checks against the CIS Microsoft Azure Foundations Benchmark. Each check evaluates specific Entra ID and Azure AD configurations — from MFA enforcement to legacy authentication blocking.

Severity-Weighted Risk Intelligence

Not all failures are equal. AVAGuard calculates organizational risk using severity-weighted scoring: Critical findings carry 10× weight while Low findings carry 1×. Your score reflects real risk, not just check counts.

CIS → NIST 800-53 → ISO 27001

Every CIS check is automatically cross-referenced to NIST 800-53 and ISO 27001 controls. Your compliance team sees exactly which regulatory requirements are at risk from a single scan.

Multi-Format Compliance Reports

Generate professional audit-ready reports in HTML, PDF, CSV, and JSON. Reports include compliance scores, per-check results, and remediation guidance — ready for stakeholder presentation.

Device Authorization & Session Control

Users authenticate through the web portal, which issues a secure device token. The platform polls for approval, receives a JWT, and maintains a heartbeat. Administrators can revoke any session in real-time.

Tamper-Proof Audit Logging

Every action — logins, scans, user changes, session revocations — is recorded in an immutable, SHA-256 hash-chained audit log. Entries cannot be modified or deleted, ensuring compliance forensics are always intact.

HOW IT WORKS

From Connection to Control

Five steps. One continuous security intelligence pipeline.

1

Connect

Securely authenticate to Microsoft Entra ID via MSAL — no agents, no installations required.

2

Analyze

Run 10 CIS benchmark checks concurrently across your tenant using ThreadPoolExecutor.

3

Score

Calculate weighted organizational risk based on severity levels: Critical 10×, High 7×, Medium 4×, Low 1×.

4

Map

Automatically cross-reference results to NIST 800-53 and ISO 27001 compliance frameworks.

5

Report
