# Troubleshooting: Incident Investigation

**Domain:** Defender for Endpoint
**Subdomain:** Incident Investigation
**Incident Type:** Troubleshooting

## Scenario / Query
How to review evidence and response details for an incident in Microsoft Defender XDR?

## Environment Context
- **Tenant Type:** Microsoft 365 Defender
- **Configuration:** Incident investigation enabled

## Symptoms
- Need to understand the verdict and remediation status of entities in an incident

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Navigate to the Evidence and Response tab in the incident
2. Review the list of supported events and suspicious entities
3. Check the verdict (Malicious, Suspicious, Clean) for each analyzed entity
4. Check the remediation status for each entity

## Validation
Verify that each entity shows a verdict and remediation status
