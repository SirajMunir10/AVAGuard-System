**Domain:** Defender for Endpoint
**Subdomain:** Microsoft Defender Antivirus
**Incident Type:** Troubleshooting

## Scenario / Query

## Root Causes
1. This error indicates that there might be a problem with your security product.

## Remediation Steps
1. Update the definitions. Either: Get your security intelligence updates in the Windows Security app.
2. Download the latest definitions from the Microsoft Security Intelligence site.
3. Run a full scan.
4. Restart the device and try again.
