# Troubleshooting: Microsoft Defender XDR

**Domain:** Defender for Endpoint
**Subdomain:** Microsoft Defender XDR
**Incident Type:** Troubleshooting

## Scenario / Query
How to review and manage automated investigations triggered by alerts in a Microsoft Defender XDR incident?

## Environment Context
- **Tenant Type:** Microsoft 365 Defender
- **Configuration:** Automated investigation configured for Defender for Endpoint and Defender for Office 365

## Symptoms
- Automated investigations are triggered by alerts in an incident
- Remediation actions may be performed automatically or wait for analyst approval

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Navigate to the Investigations tab of the incident to list all automated investigations
2. Select an investigation to view its details page for full information on investigation and remediation status
3. Check the Pending actions tab for any actions requiring approval as part of the investigation
4. Take action as part of incident remediation

## Validation
Select an investigation to navigate to its details page for full information on the investigation and remediation status
