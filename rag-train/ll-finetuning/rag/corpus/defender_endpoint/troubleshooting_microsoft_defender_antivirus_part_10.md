# Troubleshooting: Microsoft Defender Antivirus (0x80508007)

**Domain:** Defender for Endpoint
**Subdomain:** Microsoft Defender Antivirus
**Incident Type:** Troubleshooting

## Scenario / Query
How to resolve Microsoft Defender Antivirus error code 0x80508007 indicating out of memory?

## Environment Context
- **Tenant Type:** N/A
- **Configuration:** N/A

## Symptoms
- Error code 0x80508007
- Message: ERR_MP_NO_MEMORY

## Error Codes
- `0x80508007`

## Root Causes
1. This error indicates that you might have run out of memory.

## Remediation Steps
1. Check the available memory on your device.
2. Close any unused applications that are running to free up memory on your device.
3. Restart the device and run the scan again.
