## Remediation Steps
1. Select the file you submitted for deep analysis
2. Select the Deep analysis tab
3. If there are any previous reports, the report summary appears in this tab

## Validation
1. In Microsoft Defender for Endpoint, navigate to the file entity page for the file you submitted for deep analysis.
2. Select the 'Deep analysis' tab.
3. Confirm that the report summary appears, displaying previous deep analysis reports if any exist.

## Rollback
1. If the deep analysis report does not appear, verify that the file was correctly submitted for deep analysis.
2. Re-submit the file for deep analysis if necessary.
3. Wait for the analysis to complete and then check the 'Deep analysis' tab again.
