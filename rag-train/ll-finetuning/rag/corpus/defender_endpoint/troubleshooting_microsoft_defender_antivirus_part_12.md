# Troubleshooting: Microsoft Defender Antivirus (0x80508023)

**Domain:** Defender for Endpoint
**Subdomain:** Microsoft Defender Antivirus
**Incident Type:** Troubleshooting

## Scenario / Query
How to resolve error code 0x80508023 indicating a threat was not found during a scan?

## Environment Context
- **Tenant Type:** N/A
- **Configuration:** N/A

## Symptoms
- Error code 0x80508023 with message ERR_MP_THREAT_NOT_FOUND

## Error Codes
- `0x80508023`

## Root Causes
1. The threat might no longer be present on the media
2. Malware might be stopping you from scanning your device

## Remediation Steps
1. Run the Microsoft Safety Scanner
2. Update your security software and try again
