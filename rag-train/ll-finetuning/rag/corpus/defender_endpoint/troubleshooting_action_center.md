# Troubleshooting: Action Center

**Domain:** Defender for Endpoint
**Subdomain:** Action Center
**Incident Type:** Troubleshooting

## Scenario / Query
How to check activity details for actions taken on a device or file in Microsoft Defender for Endpoint Action Center?

## Environment Context
- **Tenant Type:** Microsoft 365 Defender
- **Configuration:** Action Center

## Symptoms
- Need to view details of actions such as investigation package collection, antivirus scan, app restriction, or device isolation

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Navigate to Action center in Microsoft Defender for Endpoint
2. View details including submission date/time, submitting user, and action success or failure status

## Validation
Check that all action details such as submission date/time, submitting user, and success/failure status are displayed
