6.4: Ensure adequate storage for logs (and related controls)

Overview
This policy setting specifies the maximum size of the log file in kilobytes. The maximum 
log file size can be configured between 1 megabyte (1,024 kilobytes) and 4 terabytes 
(4,194,240 kilobytes) in kilobyte increments. 
The recommended state for this setting is: Enabled: 32,768 or greater.

Audit Steps
For Ensure adequate storage for logs:
Navigate to the UI Path articulated in the Remediation section and confirm it is set as 
prescribed. This policy setting is backed by the following registry location with a 
REG_DWORD value of 32768 or greater. 
HKLM\SOFTWARE\Policies\Microsoft\Windows\EventLog\Application:MaxSize
