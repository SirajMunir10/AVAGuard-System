4.1.3.1: (L1) Ensure 'Prevent enabling lock screen camera' is set

Overview
Disables the lock screen camera toggle switch in PC Settings and prevents a camera 
from being invoked on the lock screen. 
The recommended state for this setting is: Enabled.

Why This Matters
Disabling the lock screen camera extends the protection afforded by the lock screen to 
camera features.

Audit Steps
Navigate to the UI Path articulated in the Remediation section and confirm it is set as 
prescribed. This policy setting is backed by the following registry location with a 
REG_DWORD value of 1. 
HKLM\SOFTWARE\Policies\Microsoft\Windows\Personalization:NoLockScreenCamera

Remediation
To establish the recommended configuration via configuration profiles, set the following 
Settings Catalog path to Enabled. 
Administrative Templates\Control Panel\Personalization\Prevent enabling lock 
screen camera

Key Takeaways
- Disabling the lock screen camera extends the protection afforded by the lock screen to  camera features.
