# Troubleshooting: iOS/iPadOS Enrollment (DeviceCapReached)

**Domain:** Intune
**Subdomain:** iOS/iPadOS Enrollment
**Incident Type:** Troubleshooting

## Scenario / Query
How to resolve iOS/iPadOS enrollment error: DeviceCapReached - Too many mobile devices are enrolled already

## Environment Context
- **Tenant Type:** N/A
- **Configuration:** N/A

## Symptoms
- Too many mobile devices are enrolled already error during iOS/iPadOS enrollment

## Error Codes
- `DeviceCapReached`

## Root Causes
1. User has reached the maximum number of enrolled mobile devices

## Remediation Steps
1. The user must remove one of their currently enrolled mobile devices from the Company Portal before enrolling another
2. See detailed instructions here
