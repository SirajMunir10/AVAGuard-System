# Troubleshooting: App Deployment

**Domain:** Intune
**Subdomain:** App Deployment
**Incident Type:** Troubleshooting

## Scenario / Query
How to run diagnostics for app deployment issues in Intune via the Microsoft 365 admin center?

## Environment Context
- **Tenant Type:** Microsoft 365
- **Configuration:** Intune app deployment

## Symptoms
- User having an issue with app deployment on their device

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Navigate to the Microsoft 365 admin center.
2. In the navigation pane, select Show all > Support > Help & support.
3. Alternatively, select Help & support on the bottom right side of the page.
4. Briefly describe your issue (for example, 'I need help deploying apps').
5. For the user having an issue with app deployment on their device, type their email address and then select Run tests.
6. After the diagnostic checks finish and a configuration issue is found, the system provides steps to resolve the issue.
7. If a diagnostic detects an issue and you implement a fix based on the results, consider rerunning the diagnostic to ensure the issue is completely resolved.

## Validation
Rerun the diagnostic to ensure the issue is completely resolved.
