# Troubleshooting: App Installation Troubleshooting

**Domain:** Intune
**Subdomain:** App Installation Troubleshooting
**Incident Type:** Troubleshooting

## Scenario / Query
How to run self-help diagnostics for app installation issues in Microsoft Intune?

## Environment Context
- **Tenant Type:** Commercial (not GCC High, DoD, or 21Vianet)
- **Configuration:** Intune app installation diagnostics

## Symptoms
- Administrators need help with app installation configuration
- Common tasks for which administrators request help with configuration

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Run self-help diagnostics that cover top support topics and common tasks
2. Note: These diagnostics cannot make changes to your tenant, but provide insight into known issues and instructions to fix them quickly

## Validation
1. Navigate to the Microsoft Intune admin center (https://intune.microsoft.com).
2. Go to 'Troubleshooting + support' > 'Help and support'.
3. Under 'Diagnostics', select 'App installation diagnostics'.
4. Run the diagnostic and confirm it completes without errors.
5. Review the diagnostic output to ensure it provides insight into known issues and instructions for common app installation tasks.
6. Verify that no changes were made to the tenant (as per documentation, diagnostics cannot make changes).

## Rollback
1. Since the self-help diagnostics are read-only and cannot make changes to the tenant, no rollback is required.
2. If the diagnostic fails to run, refresh the page and try again.
3. If issues persist, open a support ticket via the 'Help and support' blade in the Intune admin center.
