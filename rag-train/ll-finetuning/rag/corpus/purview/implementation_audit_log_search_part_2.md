# Implementation: Audit Log Search

**Domain:** Purview
**Subdomain:** Audit Log Search
**Incident Type:** Implementation

## Scenario / Query
How to copy search criteria from an existing audit log search job to create a new search?

## Environment Context
- **Tenant Type:** Microsoft 365
- **Configuration:** Purview audit log search criteria reuse

## Symptoms
- Need to reuse search parameters from a previous search job

## Error Codes
N/A

## Root Causes
N/A

## Remediation Steps
1. Select the existing search job from the dashboard
2. Select Copy this search on the command bar
3. The search criteria are copied to the search page
4. Modify the search criteria as needed for a new search

## Validation
Search criteria are successfully copied and available for modification on the search page
